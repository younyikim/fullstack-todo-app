## ToDo 애플리케이션

ToDo 애플리케이션을 개발하는 토이 프로젝트입니다.   
| 프론트엔드 | React.js, TypeScript, Axios, Tanstack Query(v5), Tailwind CSS, Vite | 
| --- | ---|

| 백엔드 | JavaScript, Node.js, Express.js, MongoDB | 
| --- | ---|

## Frontend
#### 설치 및 실행
1. 레포지토리 클론
    ```
    $ git clone https://github.com/younyikim/fullstack-todo-app.git
    ```
2. **`client` 디렉토리로 이동**
    ```
    cd client
    ```

3. **패키지를 설치**
    ```
    yarn
    ```

5. **서버 실행**
    ```
    yarn dev
    ```

---

## Backend
#### 설치 및 실행
1. **`backend` 디렉토리로 이동**
    ```
    cd backend
    ```

2. **패키지를 설치**
    ```
    npm install
    ```

3. **서버 실행**
    ```
    npm start
    ```
