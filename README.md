## Backend
#### 설치 및 실행
1. 레포지토리 클론
    ```
    $ git clone https://github.com/younyikim/fullstack-todo-app.git
    ```
2. **`backend` 디렉토리로 이동**
    ```
    cd backend
    ```

3. **패키지를 설치**
    ```
    npm install
    ```

5. **서버 실행**
    ```
    npm start
    ```
